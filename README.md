# Interactive Annual Report for Wellways 2020

Lanaguages:
- HTML
- CSS
- Javascript

Interactions:
- onScroll events for chapter titles, statistics, circle graphs
- onScroll highlighting on hero quotes
- svg paths for floating background shapes
